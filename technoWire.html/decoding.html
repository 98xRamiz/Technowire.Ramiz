Decoding History

Programmeertalen kan je opdelen in verschillende categorieën. De categorieën bieden de ontwikkelaars van software verschillende niveaus van abstractie en mogelijkheden om softwareoplossingen te ontwikkelen. Machine code zijn instructies die worden uitgevoerd en begrepen door een computerprocessor. De code bestaat uit binaire cijfers en nullen die acties en bewerkingen aangeven die de processor uitvoert. Assemblytalen zijn programmeertalen die een laag niveau van abstractie bieden tussen machinecodes en hogere programmeertalen. Assemblytalen maken gebruik van tekst symbolen en afkortingen. Hogere programmeertalen zijn talen die zijn ontworpen om programmeurs code te laten schrijven op een meer abstract niveau. Een voorbeeld van een hogere programmeertaal is Javascript. Vierde generatie talen ook wel 4GL genoemd. Zijn programmeertalen zijn ontworpen om de ontwikkeling van softwaretoepassingen te vergemakkelijken door middel van hogere niveaus van abstractie en automatisering. Een voorbeeld van een 4Gl taal is SQL. Objectgeoriënteerde talen ook wel OOP-talen genoemd, zijn programmeertalen die het gemakkelijker maken om complexe systemen te ontwerpen en te onderhouden. Een voorbeeld van een objectgeoriënteerde taal is Python.

HOME IT HALL OF FAME SILICON VALLEY DECODING HISTORY

C++ (1983)

Bjarne Stroustrup

C++ staat relatief dicht bij de hardware, wat veel controle geeft in termen van geheugenbeheer, performance en uiteindelijk een robuuste software ontikkeling.

Python (1991)

Guido van Rossum

Python wordt veel gebruikt bij de ontwikkeling van webapplicaties en voor data science.

HTML (1993)

Tim Berners-Lee

De belangrijkste eigenschap van HTML is dat je pagina’s en paginaelementen kan koppelen met hyperlinks. Ook vormt het de basis voor een website.

Javascript (1995)

Brendan Eich

Met javascript kan je je website interactief maken. Je kan bijvoorbeeld makkelijk animaties toevoegen.

PHP (1995)

Rasmus Lerdorf

PHP is een server gebaseerde opensource scripttaal die voornamelijk dient voor het maken van dynamische webpagina’s

Java (1996)

James Gosling

Een specifieke eigenschap van java is de overdraagbaarheid, veiligheid en prestaties.

CSS (1996)

W3C

CSS vertelt hoe een webpagina moet worden weergeven. Het geeft een stijl aan de pagina.

C# (2001)

Microsoft Corporation

C# kan je gebruiken om de moderne softwaretoepassingen van vandaag te bouwen.

Swift (2014)

Apple Inc

Swift wordt gebruikt voor het ontwikkelen van IOS, macOS en watchOS apps.
